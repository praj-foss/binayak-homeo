# Binayak Homeo
A free and open source software for Homeopathy. <br/>
It's cross-platform and made using Kotlin and JavaFX. 

## Building
- Make sure you have JDK 1.8 (or higher) installed.
- Execute this:
    ```
    gradlew build
    ```
- Get the packages from **build/distributions** folder
